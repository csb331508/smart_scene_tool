# 如何创建程序图标

## 📌 图标文件要求

您需要将图标文件命名为 `ico.ico` 并放在项目根目录：
```
d:\Documents\code\scene_tool\scen_main\ico.ico
```

## 🎨 图标规格

### 推荐格式
- **文件格式**: `.ico` (Windows 图标格式)
- **包含尺寸**: 16x16, 32x32, 48x48, 256x256 (多尺寸 ICO)
- **颜色深度**: 32位 (支持透明度)

### 为什么需要多尺寸？
- **16x16** - 文件列表小图标
- **32x32** - 文件列表大图标
- **48x48** - 桌面快捷方式
- **256x256** - Windows 7+ 大图标

## 🔧 创建方法

### 方法1: 在线转换工具（最简单）

1. **访问在线转换网站**:
   - https://www.icoconverter.com/
   - https://convertio.co/zh/png-ico/
   - https://onlineconvertfree.com/convert-format/png-to-ico/

2. **上传您的图片** (PNG/JPG/SVG)
   - 推荐上传至少 256x256 的 PNG 图片
   - 确保背景透明（如果需要）

3. **选择尺寸**:
   - 勾选: 16x16, 32x32, 48x48, 256x256
   - 或选择"包含所有尺寸"

4. **下载并重命名**:
   - 下载生成的 .ico 文件
   - 重命名为 `ico.ico`
   - 放到项目根目录

### 方法2: 使用 GIMP (免费软件)

1. **下载安装 GIMP**: https://www.gimp.org/

2. **打开您的图片**:
   ```
   文件 -> 打开 -> 选择图片
   ```

3. **调整大小**:
   ```
   图像 -> 缩放图像 -> 设置为 256x256
   ```

4. **导出为 ICO**:
   ```
   文件 -> 导出为 -> 输入文件名 "ico.ico"
   选择 "Microsoft Windows icon" 格式
   勾选多个尺寸: 16, 32, 48, 256
   导出
   ```

### 方法3: 使用 ImageMagick (命令行)

1. **安装 ImageMagick**: https://imagemagick.org/

2. **转换命令**:
   ```bash
   magick convert input.png -define icon:auto-resize=256,48,32,16 ico.ico
   ```

### 方法4: 在线 AI 生成图标

如果您没有图片，可以使用 AI 生成：

1. **访问 AI 图标生成网站**:
   - https://www.bing.com/create (Microsoft Designer)
   - https://www.canva.com/ (Canva)
   - https://icon.kitchen/ (专门的图标生成器)

2. **生成图标**:
   - 输入描述："scissors cutting film, simple icon, flat design"
     (剪刀剪视频，简单图标，扁平设计)
   - 选择样式和颜色
   - 下载 PNG 格式

3. **转换为 ICO**:
   - 使用方法1的在线转换工具

## 🎯 图标设计建议

### 主题建议
- **剪刀 + 胶片** - 表示视频剪切
- **魔法棒 + 视频** - 表示智能分割
- **时间轴 + 分段** - 表示时间分割

### 设计原则
1. **简洁明了** - 在小尺寸下也能看清
2. **有辨识度** - 与其他程序区分开
3. **专业感** - 使用合适的配色
4. **透明背景** - 更好地融入系统

### 配色建议
- **主色**: 蓝色系 (#3B82F6) - 科技感
- **辅色**: 紫色系 (#8B5CF6) - 创意感
- **强调**: 橙色系 (#F59E0B) - 活力感

## 📦 临时解决方案

如果您暂时没有图标，可以：

### 选项1: 跳过图标

```python
icon=None,  # Temporary: no icon
```

### 选项2: 使用默认 Python 图标

## ✅ 验证图标

创建图标后，可以验证：

1. **检查文件**:
   ```bash
   dir ico.ico
   ```
   应该显示文件大小（通常 10-100 KB）

2. **预览图标**:
   - 在 Windows 资源管理器中右键点击
   - 选择"属性"
   - 查看图标预览

3. **打包测试**:
   ```bash
   build.bat
   ```
   检查生成的 .exe 文件是否显示图标

## 🆘 常见问题

### Q: 图标没有显示？
A: 可能原因：
1. 文件名错误 - 必须是 `ico.ico`
2. 文件格式错误 - 必须是 `.ico` 格式
3. 路径错误 - 必须在项目根目录
4. Windows 缓存 - 重启资源管理器或重启电脑

### Q: 图标显示模糊？
A: 确保包含多种尺寸，特别是 256x256

### Q: 打包失败提示找不到图标？
A: 检查：
1. 文件确实存在
2. 文件名拼写正确
3. build.spec 中路径正确
