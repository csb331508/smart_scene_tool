## 🚀 使用说明

### 1. FFmpeg 部署
打包后，用户有两种方式使用 FFmpeg：

**方式1：放在程序旁边（推荐）**
```
场景分割工具.exe
ffmpeg.exe          ← 将 ffmpeg.exe 放在这里
```

**方式2：系统PATH**
- FFmpeg 已安装并在系统环境变量中

### 2. 运行机制
程序会自动：
1. 首先检查 `场景分割工具.exe` 所在目录是否有 `ffmpeg.exe`
2. 如果没有，检查系统 PATH
3. 如果都找不到，显示友好的错误提示

### 3. TransNetV2 权重
- ✅ 已配置打包到程序中
- ✅ 使用 `get_resource_path()` 自动定位

## 📝 技术细节

### 资源路径处理
```python
def get_resource_path(relative_path):
    try:
        # 打包后：使用 PyInstaller 的临时目录
        base_path = sys._MEIPASS
    except AttributeError:
        # 开发环境：使用脚本所在目录
        base_path = os.path.dirname(os.path.abspath(__file__))
    
    return os.path.join(base_path, relative_path)
```
