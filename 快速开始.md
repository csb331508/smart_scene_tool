# 快速开始指南

## 立即使用（3步）

### 1️⃣ 安装依赖

```bash
cd scene_segmentation_tool
pip install -r requirements.txt
```

> ⚠️ **注意**: 首次安装可能需要5-10分钟，TensorFlow约500MB

### 1️⃣ 可选：启用方案 1 的 GPU TensorFlow

如果你要让智能分析阶段也尽量使用 NVIDIA GPU，请使用单独环境，不要覆盖当前默认环境：

```bash
setup_gpu_tf_win_native.bat
run_gpu_tf_win_native.bat
```

如果下载 `cudatoolkit` / `cudnn` 时长期卡在低进度，可以尝试：

```bash
setup_gpu_tf_win_native_cn.bat
```

如果你是 RTX 50 系列等较新显卡，首次 GPU 推理可能会很慢。建议先跑一次：

```bash
warmup_gpu_tf_win_native.bat
```

这个方案固定使用：
- Python 3.10
- TensorFlow 2.10
- CUDA 11.2
- cuDNN 8.1

### 2️⃣ 安装FFmpeg

**Windows（推荐使用Chocolatey）:**
```bash
choco install ffmpeg
```

**或手动安装:**
1. 下载: https://ffmpeg.org/download.html
2. 解压到 `C:\ffmpeg`
3. 添加 `C:\ffmpeg\bin` 到系统PATH

### 3️⃣ 运行程序

```bash
python main.py
```

## 使用示例

### 智能分割单个视频

1. 启动程序: `python main.py`
2. 选择分割模式: "🤖 智能场景分割"
3. 设置时间范围: 最短5秒，最长30秒
4. 选择输入文件夹（包含视频文件）
5. 选择输出文件夹
6. 点击"开始分割"

### 批量处理多个视频

程序会自动处理输入文件夹中的所有视频文件（.mp4, .avi, .mov等），为每个视频创建单独的输出文件夹。

## 常见问题

**Q: 提示"未找到FFmpeg"怎么办？**
A: 在命令行运行 `ffmpeg -version` 检查FFmpeg是否正确安装并在PATH中。

**Q: 智能分割不可用？**
A: 确保已安装TensorFlow: `pip install tensorflow-cpu>=2.10.0`

**Q: 为什么提取片段时有 GPU，占用分析阶段却不高？**
A: 提取片段和智能分析是两条不同链路。现在抽分析帧已经优先走 FFmpeg 硬件解码，但如果 TensorFlow 仍是 CPU 版，模型推理这一步还是 CPU。

**Q: 处理速度慢？**
A: 
- 启用GPU加速（需要NVIDIA/AMD显卡）
- 或使用"普通场景分割"模式

## 配置文件

程序会自动保存配置到 `config.json`，下次启动时自动恢复上次的设置。

## 打包为独立程序

```bash
# 安装PyInstaller
pip install pyinstaller

# 运行打包脚本（Windows）
build.bat

# 或手动打包
pyinstaller build.spec
```

打包后的程序在 `dist\场景分割工具\` 文件夹中。
